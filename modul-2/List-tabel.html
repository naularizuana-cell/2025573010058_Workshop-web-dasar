Jenis-Jenis List

Teknologi yang Dipelajari

  • HTML — struktur halaman
  • CSS — tampilan halaman
    • Flexbox
    • Grid
    • Tailwind CSS
  • JavaScript — interaktivitas
  • React — library UI

Langkah Membuat Halaman Web

  1. Buat file index.html
  2. Tulis struktur HTML
  3. Tambahkan CSS untuk styling
  4. Tambahkan JavaScript untuk interaktivitas
  5. Deploy ke server

Glosarium

HTML
HyperText Markup Language — bahasa markup untuk membuat halaman web
CSS
Cascading Style Sheets — bahasa styling untuk mengatur tampilan halaman web
DOM
Document Object Model — representasi HTML sebagai struktur pohon yang bisa dimanipulasi JavaScript
